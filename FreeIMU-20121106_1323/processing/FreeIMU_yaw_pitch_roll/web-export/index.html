FreeIMU_yaw_pitch_roll

Visualize orientation information from a FreeIMU device INSTRUCTIONS: This program has to be run when you have the FreeIMU_serial program running on your Arduino and the Arduino connected to your PC. Remember to set the serialPort variable below to point to the name the Arduino serial port has in your system. You can get the port using the Arduino IDE from Tools->Serial Port: the selected entry is what you have to use as serialPort variable.
